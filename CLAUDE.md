## Project Overview

Computer vision package for electronic component (PCB pin/connector) inspection — pin detection, defect analysis, edge segmentation, and model evaluation. Built with **nbdev**: all Python code is auto-generated from Jupyter notebooks.

## Development Workflow

This is an **nbdev** project. The source of truth is the notebooks in `nbs/`, not the Python files in `new_test/`.

```bash
# Export notebooks to Python modules (after editing notebooks)
nbdev_export

# Run tests (executes all notebook cells marked for testing)
nbdev_test

# Build documentation (Quarto-based, outputs to _docs/)
nbdev_docs

# Install package in development mode
uv sync

# CI runs via: .github/workflows/test.yaml (nbdev3-ci)
# Docs deploy via: .github/workflows/deploy.yaml (quarto-ghp3)
```

## Architecture

**Notebook → Code mapping:** `nbs/XX_topic.subtopic.ipynb` → `new_test/topic/subtopic.py`
- Example: `nbs/49_patching.inference_batch_image.ipynb` → `new_test/patching/inference_batch_image.py`
- All generated Python files have `# AUTOGENERATED! DO NOT EDIT!` headers — edit the notebook instead

**Key subpackages:**
- `patching/` — Patch-based inference, edge segmentation (LightweightAttentionUNet), training, preprocessing
- `preprocessing/` — Image registration (phase correlation, Fourier-Mellin), rotation correction
- `image_gen/` — Synthetic data generation
- `model_eval/` — Model evaluation, pin extraction analysis
- `viz_utils/` — Visualization and tiling

**Local editable dependencies** (defined in `[tool.uv.sources]`):
- `cv_tools` — `/home/hasan/workspace/projects/git_data/cv_tools`
- `private_front_easy_pin_detection` — `/home/hasan/workspace/projects/git_data/private_front_easy_pin_detection`

Both are nbdev projects; run `nbdev_export` in them if their modules are missing.

## Key Details

- Python 3.12+ (currently 3.13), managed with **uv**
- Uses both PyTorch (`torch`, `timm`) and OpenCV (`opencv-python-headless`)
- ONNX export support for edge deployment
- `_modidx.py` is the nbdev-generated module index — do not edit manually
